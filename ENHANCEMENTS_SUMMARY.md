## ðŸŽ® How to Use

### Moving Elements
1. **Drag & Drop**: Click and drag any aisle to move it
2. **Snap to Grid**: Elements automatically snap to grid for alignment
3. **Visual Feedback**: Selected elements show highlight

### Drawing Boundaries
1. Click **"Rectangle"** or **"Polygon"** in Advanced Design Tools
2. Click on canvas to start drawing
3. For rectangle: Click and drag to create
4. For polygon: Click multiple points, then press `Enter` to complete
5. Press `Escape` to cancel

### 3D Floor Navigation
1. Switch to **3D view** mode
2. Use **Previous/Next** buttons to navigate floors
3. Click floor dots to jump to specific floor
4. Only selected floor renders for clarity

### Smart Alignment
1. Select multiple aisles (future feature)
2. Click alignment buttons:
   - **Align Left**: Aligns all to leftmost position
   - **Align Center**: Centers all elements
   - **Align Right**: Aligns all to rightmost position
   - **Distribute**: Evenly spaces elements

### Keyboard Shortcuts
- `Enter`: Complete polygon drawing
- `Escape`: Cancel current operation
- `Delete`/`Backspace`: Delete selected element
